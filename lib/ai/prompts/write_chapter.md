# 写章节正文

## 任务

根据提供的上下文，生成一章完整的正文草稿。

## 输入

### 小说圣经
{{storyBible}}

### 角色列表（含当前状态）
{{characters}}

### 本章目标
{{chapterGoal}}

### 分章计划
{{chapterPlan}}

### 前文摘要（最近3章）
{{previousSummaries}}

### 活跃伏笔（未回收）
{{activeForeshadowings}}

### 记忆库摘要
{{memorySummary}}

### 文风要求
{{styleGuide}}

### 目标字数
{{targetWords}}

## 输出格式

**直接输出正文内容，不要输出章节标题（如"第一章""第X章"），不要输出任何 markdown 标题标记。**

## 要求

1. 严格遵循本章目标，不跑题
2. 保持角色性格一致性
3. 适当呼应已埋伏笔
4. 结尾要有钩子，吸引读下一章
5. **字数必须达到目标字数的 90% 以上**
6. 避免 AI 味：
   - 不要用"然而""于是""就在这时"等过渡词堆砌
   - 对话要自然，不要每个人都像哲学家
   - 描写要有细节，不要泛泛而谈
   - 节奏要有变化，不要匀速推进
