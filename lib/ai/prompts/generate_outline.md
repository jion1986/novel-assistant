# 生成大纲

## 任务

根据小说设定和人设，生成全书大纲。

## 输入

小说设定：{{storyBible}}
角色列表：{{characters}}
目标字数：{{targetWords}}

## 输出格式（JSON）

```json
{
  "volumes": [
    {
      "volumeNumber": 1,
      "title": "卷标题",
      "goal": "本卷目标，100字以内",
      "chapters": [
        {
          "chapterNumber": 1,
          "title": "章节标题",
          "goal": "本章目标，50字以内",
          "keyEvents": ["关键事件1", "关键事件2"],
          "characters": ["出场角色名1", "出场角色名2"],
          "foreshadowing": ["本章埋伏笔1", "本章埋伏笔2"],
          "endingHook": "章节结尾钩子，30字以内"
        }
      ]
    }
  ]
}
```

## 要求

1. 开篇要有强钩子，吸引读者
2. 每卷要有明确的起承转合
3. 高潮位置要合理分布
4. 结尾方向要留有余地
5. 伏笔要自然植入，不突兀
6. 分卷数量根据目标字数确定：
   - 30万字以下：1-2 卷
   - 30-100万字：3-5 卷
   - 100万字以上：5 卷以上
