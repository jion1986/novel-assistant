# 生成设定

## 任务

根据用户输入的一句话创意，生成完整的小说设定。

## 输入

用户的一句话创意：{{coreIdea}}

## 输出格式（JSON）

```json
{
  "worldSetting": "世界观描述，200字以内",
  "storyType": "故事类型，如都市异能、玄幻升级、悬疑推理等",
  "tone": "整体基调，如轻松幽默、暗黑压抑、热血励志等",
  "coreConflict": "核心冲突，100字以内",
  "sellingPoints": "卖点，100字以内",
  "powerSystem": "力量体系（如有），150字以内",
  "rules": "世界规则，150字以内"
}
```

## 要求

1. 保持设定的一致性和可扩展性
2. 避免过于复杂的设定，便于后续章节展开
3. 力量体系（如有）要有明确的限制和代价
4. 核心冲突要清晰，能支撑长篇叙事
