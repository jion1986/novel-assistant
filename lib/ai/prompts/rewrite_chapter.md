# 改写章节

## 任务

根据用户要求，对章节进行风格或内容改写。

## 输入

### 原始内容
{{originalContent}}

### 改写要求
{{rewriteInstruction}}

### 可选：参考文风
{{styleReference}}

## 输出格式

直接输出改写后的正文内容。

## 改写类型

用户可能要求以下类型的改写：

1. **改文风**
   - 更像网文（节奏快、爽点密集）
   - 更克制（减少形容词，更冷静）
   - 更有情绪（加强心理描写）
   - 更有画面感（加强环境、动作描写）
   - 更简洁（删除冗余，保留核心）

2. **改人物说话**
   - 主角太油 → 降低口语化，增加稳重感
   - 配角不够稳 → 减少情绪波动，增加理性
   - 反派不够狠 → 增加威胁感和压迫感

3. **改剧情轻重**
   - 太水 → 压缩次要情节，加速主线
   - 太快 → 增加过渡和细节
   - 冲突不够 → 强化矛盾和对立

## 要求

1. 保持核心情节不变
2. 保持角色设定不变
3. 保持章节目标不变
4. 严格按照用户的改写要求执行
5. 改写后的字数与原文尽量接近（±15%）
