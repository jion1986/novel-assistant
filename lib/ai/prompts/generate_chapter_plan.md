# 生成分章计划

## 任务

根据全书大纲，为每一章生成详细的分章计划。

## 输入

### 小说设定
{{storyBible}}

### 全书大纲
{{outline}}

### 目标字数
{{targetWords}}

## 输出格式（JSON）

```json
{
  "chapters": [
    {
      "chapterNumber": 1,
      "title": "章节标题",
      "goal": "本章目标：要达成什么，推动什么",
      "characters": ["出场角色1", "出场角色2"],
      "conflict": "本章冲突点",
      "endingHook": "章节结尾钩子",
      "foreshadowing": {
        "plant": ["本章新埋伏笔"],
        "develop": ["本章发展的伏笔"],
        "resolve": ["本章回收的伏笔"]
      },
      "estimatedWords": 3000,
      "keyScenes": [
        {
          "description": "场景描述",
          "purpose": "场景作用"
        }
      ]
    }
  ]
}
```

## 要求

1. 每章必须有明确的目标和冲突
2. 章节之间要有逻辑递进关系
3. 每章结尾都要有钩子（悬念、冲突升级、新发现等）
4. 伏笔安排要自然，有埋有收
5. 单章字数控制在 2500-4000 字
6. 根据目标字数确定总章数：
   - 10万字：30-40章
   - 50万字：150-200章
   - 100万字：300-400章
